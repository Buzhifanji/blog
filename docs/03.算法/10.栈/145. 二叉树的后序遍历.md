## 理解二叉树的后序遍历

二叉树的后序遍历的记忆法则是“左右根"，即先遍历左子树节点，再遍历右子树节点，最后遍历根节点

二叉树前序遍历是“左右根"，那么结果就是：【D, E, B, F, G, C, A】

模拟过程应该是： 
- B -> D -> E 当作整体 B; 
- C -> F -> G当作整体 C;
- A -> B -> C 当作整体 A

```js
function postorderTraversal(root) {
  const result = []

  if(!root) return result

  const stack = []
  let pre = null
  while(root || stack.length) {
    while(root) {
      stack.push(root)
      root = root.left
    }
    const now = stack.pop()

    if(now.right === null || now.right === pre) {
      result.push(now.val)
      pre = now
      root = null
    } else {
      stack.push(root)
      root = root.right
    }
  }

  return result
}
```
